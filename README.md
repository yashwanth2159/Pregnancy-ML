# Early Trimester Multi-Risk Prediction of Pregnancy Complications Using Explainable Ensemble Machine Learning

##  Project Overview
This project presents an intelligent healthcare prediction system for identifying multiple pregnancy-related complications during the **Early Trimester** using **Explainable Ensemble Machine Learning**.

The system predicts maternal risk levels such as **Low Risk, Medium Risk, and High Risk** based on clinical parameters. It combines multiple machine learning algorithms to improve prediction accuracy and integrates explainable AI techniques to ensure transparency and trustworthiness.

---

##  Objectives
- Predict pregnancy complications during the early trimester.
- Improve prediction performance using ensemble learning.
- Provide interpretable results using Explainable AI.
- Support healthcare professionals in early decision-making.

---

##  Machine Learning Models Used
- Random Forest
- XGBoost
- Support Vector Machine (SVM)

### Ensemble Strategy
Voting / Stacking based ensemble model for robust predictions.

---

## Explainability Techniques
- SHAP (Global feature importance)
- LIME (Local explanation for individual predictions)

---

##  Dataset Features
The model uses maternal health-related parameters such as:

- Age
- Systolic Blood Pressure
- Diastolic Blood Pressure
- Blood Sugar
- Body Temperature
- Heart Rate

---

##  Performance Metrics
- Accuracy: **88%**
- Macro F1 Score: **0.89**
- Macro ROC-AUC: **0.97**
